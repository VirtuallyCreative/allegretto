Customizing

allegretto comes with a basic set of CSS variables:

{% highlight css %} :root { --primary-color: #1a9f60; --lightGrey-color: #ddd; --darkGrey-color: #888; --grid-maxWidth: 120rem; --grid-gutter: 1.5rem; --font-family: "Helvetica Neue", sans-serif; } {% endhighlight %}

To override the variables, just add them to your :root selector after importing retto.css

{% highlight css %} @import url(retto.css) :root { --primary-color: #da1d50; /* brand color */ --grid-maxWidth: 108rem; /* max container width 1080px */ } /* Other styles..... */ {% endhighlight %}
